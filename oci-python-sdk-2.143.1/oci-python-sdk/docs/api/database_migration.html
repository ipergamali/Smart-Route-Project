Database Migration¶

oci.database_migration.DatabaseMigrationClient Use the Oracle Cloud Infrastructure Database Migration APIs to perform database migration operations.
oci.database_migration.DatabaseMigrationClientCompositeOperations This class provides a wrapper around DatabaseMigrationClient and offers convenience methods for operations that would otherwise need to be chained together.
