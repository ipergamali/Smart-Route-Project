Optimizer¶

oci.optimizer.OptimizerClient Use the Cloud Advisor API to find potential inefficiencies in your tenancy and address them.
oci.optimizer.OptimizerClientCompositeOperations This class provides a wrapper around OptimizerClient and offers convenience methods for operations that would otherwise need to be chained together.
