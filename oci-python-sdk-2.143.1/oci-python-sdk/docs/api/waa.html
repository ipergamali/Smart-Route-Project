Waa¶

oci.waa.WaaClient API for the Web Application Acceleration service.
oci.waa.WorkRequestClient API for the Web Application Acceleration service.
oci.waa.WaaClientCompositeOperations This class provides a wrapper around WaaClient and offers convenience methods for operations that would otherwise need to be chained together.
oci.waa.WorkRequestClientCompositeOperations This class provides a wrapper around WorkRequestClient and offers convenience methods for operations that would otherwise need to be chained together.
