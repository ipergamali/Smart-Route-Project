Management Agent¶

oci.management_agent.ManagementAgentClient Use the Management Agent API to manage your infrastructure’s management agents, including their plugins and install keys.
oci.management_agent.ManagementAgentClientCompositeOperations This class provides a wrapper around ManagementAgentClient and offers convenience methods for operations that would otherwise need to be chained together.
