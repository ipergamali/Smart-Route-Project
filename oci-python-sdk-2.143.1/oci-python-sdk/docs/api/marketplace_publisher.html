Marketplace Publisher¶

oci.marketplace_publisher.AttachmentClient Use the Marketplace Publisher API to manage the publishing of applications in Oracle Cloud Infrastructure Marketplace.
oci.marketplace_publisher.MarketplacePublisherClient Use the Marketplace Publisher API to manage the publishing of applications in Oracle Cloud Infrastructure Marketplace.
oci.marketplace_publisher.OfferClient Use the Marketplace Publisher API to manage the publishing of applications in Oracle Cloud Infrastructure Marketplace.
oci.marketplace_publisher.AttachmentClientCompositeOperations This class provides a wrapper around AttachmentClient and offers convenience methods for operations that would otherwise need to be chained together.
oci.marketplace_publisher.MarketplacePublisherClientCompositeOperations This class provides a wrapper around MarketplacePublisherClient and offers convenience methods for operations that would otherwise need to be chained together.
oci.marketplace_publisher.OfferClientCompositeOperations This class provides a wrapper around OfferClient and offers convenience methods for operations that would otherwise need to be chained together.
