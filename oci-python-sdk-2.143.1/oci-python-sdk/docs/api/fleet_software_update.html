Fleet Software Update¶

oci.fleet_software_update.FleetSoftwareUpdateClient Use the Exadata Fleet Update service to patch large collections of components directly, as a single entity, orchestrating the maintenance actions to update all chosen components in the stack in a single cycle.
oci.fleet_software_update.FleetSoftwareUpdateClientCompositeOperations This class provides a wrapper around FleetSoftwareUpdateClient and offers convenience methods for operations that would otherwise need to be chained together.
