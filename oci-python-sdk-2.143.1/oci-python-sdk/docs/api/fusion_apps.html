Fusion Apps¶

oci.fusion_apps.FusionApplicationsClient Use the Fusion Applications Environment Management API to manage the environments where your Fusion Applications run.
oci.fusion_apps.FusionApplicationsClientCompositeOperations This class provides a wrapper around FusionApplicationsClient and offers convenience methods for operations that would otherwise need to be chained together.
