__init__(**kwargs)¶

Initializes a new Migration object with values from keyword arguments. This class has the following subclasses and if you are using this class as input to a service operations then you should favor using a subclass over the base class:

  • OracleMigration
  • MySqlMigration

The following keyword arguments are supported (corresponding to the getters/setters of this class):

Parameters:
  • id (str) – The value to assign to the id property of this Migration.
  • description (str) – The value to assign to the description property of this Migration.
  • database_combination (str) – The value to assign to the database_combination property of this Migration. Allowed values for this property are: “MYSQL”, “ORACLE”, ‘UNKNOWN_ENUM_VALUE’. Any unrecognized values returned by a service will be mapped to ‘UNKNOWN_ENUM_VALUE’.
  • display_name (str) – The value to assign to the display_name property of this Migration.
  • compartment_id (str) – The value to assign to the compartment_id property of this Migration.
