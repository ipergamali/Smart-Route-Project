Golden Gate¶

oci.golden_gate.GoldenGateClient Use the Oracle Cloud Infrastructure GoldenGate APIs to perform data replication operations.
oci.golden_gate.GoldenGateClientCompositeOperations This class provides a wrapper around GoldenGateClient and offers convenience methods for operations that would otherwise need to be chained together.
