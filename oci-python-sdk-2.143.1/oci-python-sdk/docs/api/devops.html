Devops¶

oci.devops.DevopsClient Use the DevOps API to create DevOps projects, configure code repositories, add artifacts to deploy, build and test software applications, configure target deployment environments, and deploy software applications.
oci.devops.DevopsClientCompositeOperations This class provides a wrapper around DevopsClient and offers convenience methods for operations that would otherwise need to be chained together.
