Data Safe¶

oci.data_safe.DataSafeClient APIs for using Oracle Data Safe.
oci.data_safe.DataSafeClientCompositeOperations This class provides a wrapper around DataSafeClient and offers convenience methods for operations that would otherwise need to be chained together.
