Artifacts¶

oci.artifacts.ArtifactsClient API covering the Artifacts and [Registry](/iaas/Content/Registry/Concepts/registryoverview.htm) services.
oci.artifacts.ArtifactsClientCompositeOperations This class provides a wrapper around ArtifactsClient and offers convenience methods for operations that would otherwise need to be chained together.
