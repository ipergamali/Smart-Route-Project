Network Firewall¶

oci.network_firewall.NetworkFirewallClient Use the Network Firewall API to create network firewalls and configure policies that regulates network traffic in and across VCNs.
oci.network_firewall.NetworkFirewallClientCompositeOperations This class provides a wrapper around NetworkFirewallClient and offers convenience methods for operations that would otherwise need to be chained together.
