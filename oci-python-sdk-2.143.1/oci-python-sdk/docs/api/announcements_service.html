Announcements Service¶

oci.announcements_service.AnnouncementClient Manage Oracle Cloud Infrastructure console announcements.
oci.announcements_service.AnnouncementSubscriptionClient Manage Oracle Cloud Infrastructure console announcements.
oci.announcements_service.AnnouncementsPreferencesClient Manage Oracle Cloud Infrastructure console announcements.
oci.announcements_service.ServiceClient Manage Oracle Cloud Infrastructure console announcements.
oci.announcements_service.AnnouncementClientCompositeOperations This class provides a wrapper around AnnouncementClient and offers convenience methods for operations that would otherwise need to be chained together.
oci.announcements_service.AnnouncementSubscriptionClientCompositeOperations This class provides a wrapper around AnnouncementSubscriptionClient and offers convenience methods for operations that would otherwise need to be chained together.
oci.announcements_service.AnnouncementsPreferencesClientCompositeOperations This class provides a wrapper around AnnouncementsPreferencesClient and offers convenience methods for operations that would otherwise need to be chained together.
oci.announcements_service.ServiceClientCompositeOperations This class provides a wrapper around ServiceClient and offers convenience methods for operations that would otherwise need to be chained together.
