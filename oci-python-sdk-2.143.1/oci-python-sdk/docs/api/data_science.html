Data Science¶

oci.data_science.DataScienceClient Use the Data Science API to organize your data science work, access data and computing resources, and build, train, deploy and manage models and model deployments.
oci.data_science.DataScienceClientCompositeOperations This class provides a wrapper around DataScienceClient and offers convenience methods for operations that would otherwise need to be chained together.
