Resource Manager¶

oci.resource_manager.ResourceManagerClient Use the Resource Manager API to automate deployment and operations for all Oracle Cloud Infrastructure resources.
oci.resource_manager.ResourceManagerClientCompositeOperations This class provides a wrapper around ResourceManagerClient and offers convenience methods for operations that would otherwise need to be chained together.
