Media Services¶

oci.media_services.MediaServicesClient Media Services (includes Media Flow and Media Streams) is a fully managed service for processing media (video) source content.
oci.media_services.MediaStreamClient Media Services (includes Media Flow and Media Streams) is a fully managed service for processing media (video) source content.
oci.media_services.MediaServicesClientCompositeOperations This class provides a wrapper around MediaServicesClient and offers convenience methods for operations that would otherwise need to be chained together.
oci.media_services.MediaStreamClientCompositeOperations This class provides a wrapper around MediaStreamClient and offers convenience methods for operations that would otherwise need to be chained together.
