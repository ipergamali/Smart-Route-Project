Data Labeling Service Dataplane¶

oci.data_labeling_service_dataplane.DataLabelingClient Use Data Labeling API to create Annotations on Images, Texts & Documents, and generate snapshots.
oci.data_labeling_service_dataplane.DataLabelingClientCompositeOperations This class provides a wrapper around DataLabelingClient and offers convenience methods for operations that would otherwise need to be chained together.
