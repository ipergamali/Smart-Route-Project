Disaster Recovery

oci.disaster_recovery.DisasterRecoveryClient Use the Full Stack Disaster Recovery (DR) API to manage disaster recovery for business applications.
oci.disaster_recovery.DisasterRecoveryClientCompositeOperations This class provides a wrapper around DisasterRecoveryClient and offers convenience methods for operations that would otherwise need to be chained together.
