Identity Domains

oci.identity_domains.IdentityDomainsClient Use the Identity Domains API to manage resources within an identity domain, for example, users, dynamic resource groups, groups, and identity providers.
oci.identity_domains.IdentityDomainsClientCompositeOperations This class provides a wrapper around IdentityDomainsClient and offers convenience methods for operations that would otherwise need to be chained together.
