Logging¶

oci.logging.LoggingManagementClient Use the Logging Management API to create, read, list, update, move and delete log groups, log objects, log saved searches, and agent configurations.
oci.logging.LoggingManagementClientCompositeOperations This class provides a wrapper around LoggingManagementClient and offers convenience methods for operations that would otherwise need to be chained together.
