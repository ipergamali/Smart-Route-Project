routing_method

Gets the routing_method of this UpdateConnectionDetails. Controls the network traffic direction to the target: SHARED_SERVICE_ENDPOINT: Traffic flows through the Goldengate Services network to public hosts. Cannot be used for private targets. SHARED_DEPLOYMENT_ENDPOINT: Network traffic flows from the assigned deployments private endpoint through the deployments subnet. DEDICATED_ENDPOINT: A dedicated private endpoint is created in the target VCN subnet for the connection. The subnetId is required when DEDICATED_ENDPOINT networking is selected.

Allowed values for this property are: SHARED_SERVICE_ENDPOINT, SHARED_DEPLOYMENT_ENDPOINT, DEDICATED_ENDPOINT

Returns:The routing_method of this UpdateConnectionDetails.
