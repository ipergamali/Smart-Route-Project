oci.opensearch.OpensearchClusterClient The OpenSearch service API provides access to OCI Search Service with OpenSearch.
oci.opensearch.OpensearchClusterBackupClientCompositeOperations This class provides a wrapper around OpensearchClusterBackupClient and offers convenience methods for operations that would otherwise need to be chained together.
