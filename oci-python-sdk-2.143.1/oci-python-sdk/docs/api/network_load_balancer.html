Network Load Balancer¶

oci.network_load_balancer.NetworkLoadBalancerClient This describes the network load balancer API.
oci.network_load_balancer.NetworkLoadBalancerClientCompositeOperations This class provides a wrapper around NetworkLoadBalancerClient and offers convenience methods for operations that would otherwise need to be chained together.
