Desktops¶

oci.desktops.DesktopServiceClient Create and manage cloud-hosted desktops which can be accessed from a web browser or installed client.
oci.desktops.DesktopServiceClientCompositeOperations This class provides a wrapper around DesktopServiceClient and offers convenience methods for operations that would otherwise need to be chained together.

Models¶

oci.desktops.models.ChangeDesktopPoolCompartmentDetails Provides the OCID of the new compartment that will contain the desktop pool.
oci.desktops.models.CreateDesktopPoolDesktopSessionLifecycleActions The details of action to be triggered in case of inactivity or disconnect
oci.desktops.models.CreateDesktopPoolDetails Provides the configuration information used to create the desktop pool.
oci.desktops.models.CreateDesktopPoolPrivateAccessDetails The details of the desktop’s private access network connectivity to be set up for the desktop pool.
oci.desktops.models.CreateDesktopPoolShapeConfigDetails The compute instance shape configuration requested for each desktop in the desktop pool.
oci.desktops.models.Desktop Provides information about a desktop including name, state, id, configuration, owner, and time created.
oci.desktops.models.DesktopAvailabilityPolicy Provides the start and stop schedule information for desktop availability of the desktop pool.
oci.desktops.models.DesktopCollection A list of desktops.
oci.desktops.models.DesktopDevicePolicy Provides the settings for desktop and client device options, such as audio in and out, client drive mapping, and clipboard access.
oci.desktops.models.DesktopImage Provides information about the desktop image.
oci.desktops.models.DesktopNetworkConfiguration Provides information about the network configuration of the desktop pool.
oci.desktops.models.DesktopPool Provides information about a desktop pool including all configuration parameters.
oci.desktops.models.DesktopPoolCollection A list of desktop pools.
oci.desktops.models.DesktopPoolDesktopCollection A list of desktops within a desktop pool.
oci.desktops.models.DesktopPoolDesktopSummary Provides information about a desktop within a desktop pool, such as if it is assigned to a user, the state, owner, desktop pool, resource id, and time created.
oci.desktops.models.DesktopPoolPrivateAccessDetails The details of the desktop’s private access network connectivity that were used to create the pool.
oci.desktops.models.DesktopPoolShapeConfig The shape configuration used for each desktop compute instance in the desktop pool.
oci.desktops.models.DesktopPoolSummary Provides information about the desktop pool such as number of active desktops, name, OCID, state, and maximum size.
oci.desktops.models.DesktopPoolVolumeCollection A list of volumes within a desktop pool.
oci.desktops.models.DesktopPoolVolumeSummary Provides information about a volume within the desktop pool.
oci.desktops.models.DesktopSchedule Provides the schedule information for a desktop.
oci.desktops.models.DesktopSessionLifecycleActions Action to be triggered on inactivity or disconnect
oci.desktops.models.DesktopSummary Provides information about a desktop.
oci.desktops.models.DisconnectConfig Action and grace period for disconnect
oci.desktops.models.HostingOptions Provides information about where a desktop is hosted.
oci.desktops.models.InactivityConfig Action and grace period for inactivity
oci.desktops.models.UpdateDesktopDetails Provides the details of an updated desktop.
oci.desktops.models.UpdateDesktopPoolDesktopSessionLifecycleActions The update details of action to be triggered in case of inactivity or disconnect
oci.desktops.models.UpdateDesktopPoolDetails Provides the details of a request to update the desktop pool.
oci.desktops.models.WorkRequest A description of workrequest status.
oci.desktops.models.WorkRequestError An error encountered while executing a work request.
oci.desktops.models.WorkRequestErrorCollection Results of a workRequestError search.
oci.desktops.models.WorkRequestLogEntry A log message from the execution of a work request.
oci.desktops.models.WorkRequestLogEntryCollection Results of a workRequestLog search.
