Integration¶

oci.integration.IntegrationInstanceClient Oracle Integration API.
oci.integration.IntegrationInstanceClientCompositeOperations This class provides a wrapper around IntegrationInstanceClient and offers convenience methods for operations that would otherwise need to be chained together.
