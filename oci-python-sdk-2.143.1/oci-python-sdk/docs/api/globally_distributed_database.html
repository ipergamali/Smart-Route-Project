Globally Distributed Database¶

oci.globally_distributed_database.ShardedDatabaseServiceClient Use the Globally Distributed Database service APIs to create and manage distributed databases.
oci.globally_distributed_database.ShardedDatabaseServiceClientCompositeOperations This class provides a wrapper around ShardedDatabaseServiceClient and offers convenience methods for operations that would otherwise need to be chained together.
